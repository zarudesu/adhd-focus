## Feature Unlocks

### Navigation Items (Sidebar)

Features unlock progressively as users demonstrate readiness:

```
Day 1-2: Getting Started
==========================

[INBOX] -----------------> Always Available
    |
    +---> Add 1 task -----> [INBOX COUNT] shows task count
    |
    +---> Add 3 tasks ----> [ACHIEVEMENTS] appears
    |
    +---> Move to today --> [TODAY] appears
    |
    +---> Complete 1 -----> [COMPLETED] appears


Day 3-7: Building Habits
==========================

    +---> Complete 3 -----> [DAILY CHECKLIST] appears
    |
    +---> Schedule future -> [SCHEDULED] appears
    |
    +---> Complete 5 -----> [FOCUS MODE] appears
    |
    +---> Add 10 tasks ---> [PROJECTS] appears
    |
    +---> Level 5 --------> [CREATURES] appears


Week 2+: Power User
==========================

    +---> Complete 10 ----> [QUICK ACTIONS] appears
    |
    +---> 7-day streak ---> [STATISTICS] appears
```

## XP Formula

```
XP required for level = floor(100 * (level ^ 1.5))

Level 1 -> 2:   100 XP
Level 2 -> 3:   283 XP
Level 5 -> 6:   1,118 XP
Level 10 -> 11: 3,162 XP
Level 20 -> 21: 8,944 XP
Level 50 -> 51: 35,355 XP
```

## Achievement XP Scaling

```typescript
function calculateXP(count: number, multiplier: number = 1): number {
  if (count <= 1) return Math.round(5 * multiplier);
  if (count <= 5) return Math.round(10 * multiplier);
  if (count <= 10) return Math.round(15 * multiplier);
  if (count <= 25) return Math.round(25 * multiplier);
  if (count <= 50) return Math.round(40 * multiplier);
  if (count <= 100) return Math.round(75 * multiplier);
  if (count <= 250) return Math.round(125 * multiplier);
  if (count <= 500) return Math.round(200 * multiplier);
  if (count <= 1000) return Math.round(350 * multiplier);
  if (count <= 2500) return Math.round(500 * multiplier);
  if (count <= 5000) return Math.round(750 * multiplier);
  if (count <= 10000) return Math.round(1000 * multiplier);
  return Math.round(1500 * multiplier);
}
```

**Multipliers:**
- Critical priority: 1.5x
- Daily achievements: 1.5x
- Weekly achievements: 1.3x
- Hourly achievements: 2x
- Speed achievements: 2.5x
- Combo achievements: 1.3x
- Triple combos: 1.5x
